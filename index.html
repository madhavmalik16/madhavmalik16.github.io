Projects

Analysis of Iris Dataset

Python, Machine Learning

Iris dataset divides the flower in three classes namely Setosa, Versicolor, Virginica based on four attributes namely Sepal Length, Sepal Width, Petal Length, Petal Width. The given dataset was shuffled randomly and divided into train-test datasets with 75-25 ratio. 10 different classifiers were executed and the results were then combined and using ensembling techniques combination of different models were run on the dataset. The accuracy of predicting the new data point came out to be nearly 97%.
